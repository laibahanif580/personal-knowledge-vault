# Personal Knowledge Vault

A modern dark-themed Second Brain application built with React, TypeScript, Vite, and TanStack Router.

Personal Knowledge Vault helps users organize notes, connect ideas, manage knowledge visually, and maintain a centralized workspace for personal productivity.

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/personal-knowledge-vault.git
```

Navigate to the project:

```bash
cd personal-knowledge-vault
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:8080
```

---

## Build for Production

```bash
npm run build
```

---

## Deployment

### Vercel

This project is fully compatible with Vercel.

Recommended settings:

```text
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```

Deploy by:

1. Push repository to GitHub
2. Import repository into Vercel
3. Deploy

No additional configuration is required.
